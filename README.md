ゲームテスト時注意
基本操作

キーボード
A：左移動
D：右移動
Z：コンクリート発射
Shift：コンクリート種類変更
Space：ジャンプ

コントローラー(xboxone)
左スティック : 左右移動
Bキー : コンクリート発射
Xキー : コンクリート種類変更
Yキー : カメラのズームアウト
Aキー : ジャンプ
